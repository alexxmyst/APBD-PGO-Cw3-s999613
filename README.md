# Projekt Kameralnej Kawiarni

Projekt aplikacji konsolowej dla kawiarni. Pozwala rejestrować produkty z menu, dane klientów i pojedyńcze zamówienia. 

Program umożliwia:
 - utworzenie kilku produktów dostępnych w menu,
 - utworzenie klienta,
 - zbudowanie zamówienia zawierającego listę produktów,
 - obliczenie łącznej wartości zamówienia,
 - wypisanie czytelnego podsumowania w konsoli,
 - porównanie wybranych obiektów przy użyciu equals().
